# Super Auction

WoW addon for streamlined auction house workflow.

## Features

- **Expansion Filter** - Defaults auction house and crafting order searches to the current expansion
- **Spacebar Posting** - Post auctions with the Space key while the sell tab is open
- **Paste & Shift-Click Search** - Automatically triggers an auction house search when pasting or shift-clicking an item into the search bar
- **Favorites Sync** - Save and import auction house favorites across characters via account-wide storage
